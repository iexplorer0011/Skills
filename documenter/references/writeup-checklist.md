# Writeup Checklist

## Mandatory Sections

- Summary
- Vulnerability Analysis
- Exploit Script
- Flag/Proof

## Evidence Quality Gates

- Include exact binary/libc identifiers.
- Include key offsets and gadget sources.
- Include at least one reproducible run command.
- Include direct success proof (shell transcript or flag output).

## Vulnerability Analysis Gates

- State vulnerability class unambiguously.
- Explain trigger condition with concrete evidence.
- Explain mitigation impact and bypass logic.

## Exploit Script Gates

- Match script content with final working version.
- Avoid stale payload snippets.
- Confirm script narrative matches actual execution order.

## Final Sanity Check

- Ensure a new reader can reproduce the result without hidden assumptions.
- Remove irrelevant debugging noise from proof transcript.
